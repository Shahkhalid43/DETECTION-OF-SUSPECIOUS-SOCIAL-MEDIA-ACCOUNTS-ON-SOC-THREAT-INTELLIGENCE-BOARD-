# DETECTION-OF-SUSPECIOUS-SOCIAL-MEDIA-ACCOUNTS-ON-SOC-THREAT-INTELLIGENCE-BOARD-
Detects fake/bot social media accounts (Twitter, Facebook, LinkedIn) using a Random Forest classifier trained on behavioral features like follower ratio, activity score, and profile completeness. Feeds results into a SOC-style Threat Intelligence Board for analyst review. Built with Python &amp; scikit-learn.
